# Changelog

All notable changes to EdgeCase Equalizer will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-10

### Added
- Complete practice management system for independent therapists
- SQLCipher-encrypted database for client data protection
- Entry-based client files (Profile, Session, Communication, Absence, Item, Upload)
- Statement generation with PDF output and email composition
- Guardian billing with automatic payment splits
- Couples/family/group therapy support with per-member fee allocation
- Income and expense tracking with financial reports
- Calendar integration (.ics export and AppleScript for macOS Calendar)
- Optional local LLM integration for clinical note assistance (AI Scribe)
- Automated backup system (full and incremental)
- Edit history tracking for compliance
- 41 automated tests for critical business logic
- Comprehensive documentation
